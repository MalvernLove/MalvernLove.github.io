Excel 汇率速查（可联网更新）

尽管微软在 Windows 11 自带的计算器加入了货币兑换的计算，各大搜索引擎和语音助手也很方便，但我总感觉用起来没那么方便。

API

这里我在网上随便找了两个 API，各国汇率 https://api.nxvav.cn/api/exchange-rate/ 和比特币汇率 https://coinyep.com/api/v1/rates/BTC/CNY/1，具体使用方法可以在他们官网当日货币汇率 | nuoxian’s API 使用文档和 CoinYEP - 货币换算上找到。

Excel

其实一开始是想用 Excel 的 VBA 来实现的，但后面发现 Excel 可以通过网页获取数据（如下图），那么事情就简单多了。

通过网页获取数据

在线获取汇率完成后，基准货币乘上相对应的汇率就行，这个就看大家常用哪些货币了。我制作了一份比较适合我的，在 Malvern's File Server 可自取。Ctrl+Alt+F5 可以更新汇率。

A1 单元格可供修改，意为基准货币数值。第 3 行是基准货币类别，第 1 列（A 列）是目标货币类别。比如要想知道 100 元人民币可以兑换多少美元，步骤如下：

  1. 在 A1 单元格输入 100。
  2. 在第 3 行找到 “人民币”，并看准在哪一列（C 列）。
  3. 在第 1 列（A 列）找到 “美元”，并看准在哪一行（第 6 行）。
  4. 找到上述两步对应行和列的单元格（C6），即为答案（14.0598）。
