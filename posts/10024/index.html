Solidworks快捷键

标准Windows快捷键

  1. Ctrl + O：打开文件
  2. Ctrl + S：保存文件
  3. Ctrl + Z：撤回
  4. Ctrl + C：复制
  5. Ctrl + V：粘贴
  6. Alt + 数字：插入标准Windows符号（如下例）
  7. Alt + 0176：插入°
  8. Alt + 0216：插入Ø
  9. Alt + 0181：插入µ
  10. 详细请见：
    https://www.cnblogs.com/letisl/p/12372408.html
    https://www.cnblogs.com/letisl/p/12372408.html
    https://unicode-table.com/cn/
    https://unicode-table.com/cn/

用户界面快捷键

  1. Ctrl + Q：强制重新生成
  2. Ctrl + Tab：在打开的SolidWorks文件之间切换文件
  3. Enter：重复上一命令

显示快捷键

  1. Alt + 方向键：顺时针或者逆时针旋转
  2. Alt + 鼠标中键： 平行视图平面旋转模型
  3. Shift + 方向键：水平或者垂直旋转模型90°
  4. Shift + Z：放大模型
  5. Z：缩小模型
  6. F：整屏显示模型
  7. G：放大镜
  8. 空格键：视图定向菜单

选择快捷键

  1. Ctrl + 鼠标左键：选择多个
  2. Shift + 鼠标左键：选择FeatureManager设计树中两个选定项目之间的所有特征

移动快捷键

  1. Ctrl + 鼠标中键：移动模型
  2. Shift + 鼠标左键拖拽：
  • 草图模式下会移动一组草图实体。
  • 零件模式下会移动特征
  • 工程图模式下将选定的图形视图和任何相关视图一起移动，就好像它们是一个实体一样。

草绘快捷键

  1. Ctrl + 左键拖拽：拖动端点时抑制推理线
  2. Shift + 左键拖拽：当绘制一条线时，该线将按照特定的长度进行捕捉
  3. Tab ：在3D草图中绘制草图时，改变XYZ平面

装配体快捷键

  1. Alt + 鼠标左键拖拽：
  • 复制组件并创建智能配合
  • 重新排序组件。 在FeatureManager设计树中移动组件时，确保组件在树中保持在同一级别，并防止组件移动到子组件中
  1. Ctrl + 鼠标左键拖拽：
  • 复制组件
  • 在具有预选组件的组合中，当按Ctrl拖动选择框时，会逆转选择
  • 在将工具箱组件拖动到装配体中时，创建零件而不是配置。
  1. Tab：
  • 隐藏指针下面的所有组件
  • 插入组件时。 旋转组件90º
  1. Shift + Tab：
  • 显示指针下面的所有组件
  • 插入组件时，旋转组件90º
  1. Ctrl + Shift + Tab ：暂时将所有隐藏的组件显示为透明，并让您选择要显示的组件
