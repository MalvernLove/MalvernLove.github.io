WSL2——体验Ubuntu

安装

直接在微软商店（Microsoft Store）搜索Ubuntu，然后选一个喜欢的版本下载就行了，我这里下载的是Ubuntu-Preview。需要注意，如果下LTS版本的，是不能升级大版本的，只会永久停留在当前本版本。

最好在安装后执行，不然可能会缺少东西。

wsl --update

换源

由于众所周知的长城防火墙原因，一般进入Ubuntu的第一件事情就是换国内的镜像源，比如清华、北邮、中科大等。如果没有vim，可以改用vi，操作是一样的。

sudo vim /etc/apt/sources.list

然后改成镜像源的地址，这里需要一点vi/vim的操作能力，如果不会的话可以改用nano或者gedit。

https://mirrors.tuna.tsinghua.edu.cn/help/ubuntu/
https://mirrors.tuna.tsinghua.edu.cn/help/ubuntu/

中文环境

  1. 安装中文支持

    sudo apt install language-pack-zh-han*

  2. 改地域
    这里选择zh_CN.UTF-8。

    sudo dpkg-reconfigure locales

  3. 改地域配置

    sudo vim /etc/default/locale

全部删除，然后改成下面的，注意不能有多余的空格。

LANG="zh_CN.UTF-8"
LANGUAGE="zh_CN:zh"
LC_NUMERIC="zh_CN"
LC_TIME="zh_CN"
LC_MONETARY="zh_CN"
LC_PAPER="zh_CN"
LC_NAME="zh_CN"
LC_ADDRESS="zh_CN"
LC_TELEPHONE="zh_CN"
LC_MEASUREMENT="zh_CN"
LC_IDENTIFICATION="zh_CN"
LC_ALL="zh_CN.UTF-8"
  1. 改环境
    sudo vim /etc/environment

另起一行，添加以下内容，注意不能有多余的空格。

LANG="zh_CN.UTF-8"
LANGUAGE="zh_CN:zh"
LC_NUMERIC="zh_CN"
LC_TIME="zh_CN"
LC_MONETARY="zh_CN"
LC_PAPER="zh_CN"
LC_NAME="zh_CN"
LC_ADDRESS="zh_CN"
LC_TELEPHONE="zh_CN"
LC_MEASUREMENT="zh_CN"
LC_IDENTIFICATION="zh_CN"
LC_ALL="zh_CN.UTF-8"

  1. 改时区

    sudo hwclock -s
sudo apt-get install tzdata
sudo dpkg-reconfigure tzdata

    最后选择Asia/Shanghai时区。

  2. 中文字体
    同步Windows字体。

    sudo apt install fontconfig
sudo vim /etc/fonts/local.conf

    在下面添加

    <?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "fonts.dtd">
<fontconfig>
 <dir>/mnt/c/Windows/Fonts</dir>
</fontconfig>

    刷新字体缓存

    sudo fc-cache -fv

  3. 中文输入法

    sudo apt install dbus-x11 fcitx im-config fcitx-sunpinyin

    然后Windows开始菜单打开fcitx，添加pinyin，再执行下面代码。

    sudo im-config -n fcitx
sudo vim /etc/locale.gen
sudo vim  ~/.profile

    在下面添加

    export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS=@im=fcitx
export DefaultIMModule=fcitx
fcitx-autostart &>/dev/null
source ~/.profile
