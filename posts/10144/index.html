中国风味的互联网小指南

深感痛心于大家受简中互联网之刁难。故在此分享一些我是如何在缺乏互联网精神（极度封闭）的互联网下冲浪之经验与感受。
其实在我看来，国内互联网厂商出品的软件，绝大多数都是垃圾软件。

基本原则

  1. 隐私保护 ：尽量不下载客户端，能使用网页端就使用网页端。手机则受限于移动互联网的天然不友好性，又处于中国网络环境，则尽量使用微信小程序。
  2. 不受限 “冲浪” ：不要用第三方的浏览器，Windows 自带的 Edge 和 Mac 自带的 Safari 已经足够优秀了，如果实在不习惯的话，建议用 Firefox 或者 Chrome。
  3. 拒绝 “李鬼” ：换掉搜索引擎吧，如今的百度搜索体验已经非常差了。有条件的使用谷歌，没条件用必应也可以。

Windows

  1. 不要迷信所有修改过的 Windows 系统，直接去 MSDN 下载最新的镜像。另外，除非明确知道所使用的软件 / 硬件与 Windows 11 哪里不兼容，否则请用 Windows 11，Windows 10 和 Windows 7 已经无法提供支持了。
  2. 固态硬盘分区会使得体验下降，不懂电脑的话，一整个 C 盘没有弊端。固态硬盘的寿命与其总写入量有关，分盘后，从一个分区移动文件到另一个分区会产生实际写入，而在同一个分区内移动文件是没有实际写入的，所以分盘会使固态硬盘的寿命降低。补充一下，机械硬盘的寿命则与其通电时长有关，和写入量没啥关系，因此对于用有机械硬盘的电脑，不用的时候睡眠 / 关机就行了。
  3. 微软自家的安全软件已经很好了，不要再用额外的安全软件了。一个良好的网络意识胜过所有的安全软件。
  4. 不推荐任何第三方的软件管家。下载软件时，可以先去微软商店（Microsoft Store）看看有没有，没有的话去官网下载。后续如果软件有更新的话，软件本身会有提示的。实在有需求的话，建议去搜索 “winget” 了解一下。
  5. 卸载软件怕有残留？如果用的是微软商店下载的话，卸载后都是很干净的。实在怕有残留的话，可以用 GeekUninstaller。
  6. 偶尔想清理垃圾的话，在文件管理器右击 C 盘会有一个清理选项，点开后在按清理系统垃圾，可以根据自己需求勾选清理内容（一般全选即可）。

移动设备

Android

  1. 安卓是最受荼毒的，建议尽量从 Google Play 商店下载 app。尽量也不要给 app 后台权限。
  2. 不过由于国行手机的特殊性，建议备一个第三方浏览器（推荐 Firefox 全球版或者 Edge），以确保尽可能不受限的互联网访问。
  3. 安卓因为基于 Linux，文件目录结构也很相似。现在较新的 app 由于谷歌的规定，也都遵守文件放置规则，微信等即时通信 app 保存的文件（副本）一般在根目录下的 Download 文件夹。DCIM 文件夹是手机拍摄的照片和视频，Pictures 文件夹是 app 保存的图片。尽管如此，仍然有很多 app 在文件目录里乱放文件，这个也没有特别好的办法。
  4. 其实除了开热点外，还有两种方法能够共享网络。一是手机通过 USB 连接电脑时，如果想给电脑共享网络，可以打开 USB 共享网络。这相当于电脑插网线上网，这样不仅更省电（手机还能充电），网络连接质量也更好。二是蓝牙共享网络，这个一般用在车机上，避免了手机长时间开热点而发烫，不过蓝牙共享的网络网速较慢，但应付开车就正好了。
  5. 善用蓝牙。如果笔记本电脑没有网络连接，又想从手机发送文件到电脑上，多数人会选择开热点，再使用通讯软件传，也会人会用数据线传。在我看来，这样操作比较复杂，还不优雅。我觉得一个比较好的方法是使用手机蓝牙发送文件，一般几 MB 的文件传输的时间还是很快的。

iOS/iPad OS

因为苹果的封闭性，iOS/iPad OS 无需特别的建议。

其他桌面系统

macOS

因为苹果的封闭性，macOS 同样无需特别的建议，尽量从 App Store 下软件就行。

Linux

都能用 Linux 了，也就不必看这篇文章了。

软件推荐

  1. 手机连接：这个是 Windows 11 自带的软件，功能挺多的。主要可以共享剪切板（文本和图片），还能看通知和传文件。
  2. Firefox：算是为数不多的非 Chromium 内核浏览器了，主要是能够所有客户端同步历史记录和密码等。
  3. Everything：快速搜索文件，个人电脑上用处不大，工作电脑上非常好用。
  4. Tailscale：这个能创建一个虚拟局域网，‍使得不同设备互联。
  5. 咖啡因：这个是安卓 app，打开后能使屏幕常亮。
  6. 冰箱｜小黑屋：这两个都是安卓 app，能冻结 app。我用在冻结国产的流氓 app，比如淘宝、京东、美团和中国移动等。
  7. LocalSend：这个是局域网快速传输的，比如我想把文件从安卓手机传到 iPad，用这个就很快。
  8. Maccy：这个是 macOS 软件，因为 macOS 没有像 Windows 那样方便的历史剪切板（Win + V），这个软件还挺有用的。
  9. Keep Screen On：这是一个网页，启用后能屏幕常亮，所有系统都能用。
  10. OneDrive：多端同步美滋滋，美中不足的是网页端被墙了。
  11. SpaceDesk：适用于桌面端，能把几乎所有设备变成副屏（不在同一网络可用上边的 Tailscale 或者用网线串联）。主设备安装 Driver，副设备安装 Viewer。
  12. SingleFile：能够把网页保存单个文件（*.html）的浏览器拓展，可以在浏览器拓展商店下载到。

题外话

  • 不少人都吐槽国外品牌的手机标配储存空间小，但在我看来，这完全就是国内互联网厂商的问题。大家可以比对一下，国内厂商出品的软件体积都比同类国外软件大很多，下载打开后，里面的内容也充斥着各类广告，疯狂占用存储空间（尤其是微信🤬）。
  • 数据无价。我认为数码设备中最重要的是我们用户的数据，比如拍摄的照片视频、喜欢的音乐和个人的一些文件。我建议这些重要的内容备份在不同的设备上，比如我的照片和视频除了手机，会在 NAS 和笔记本电脑上不定时备份，以防丢失。
  • 从节省空间的角度讲，可以尝试使用 heif 和 hevc 的格式拍摄照片和视频。现在绝大部分设备和软件都支持了这个格式了。
  • 我一般会尽可能地使用开源软件，我认为开源是互联网精神的最佳体现。
  • 有条件的建议备一张境外的 SIM 卡，用于在某些严苛的网络环境下突破封锁。
